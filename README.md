# LibreriaOnline

LibreriaOnline es un proyecto en desarrollo que busca ofrecer una plataforma para la gestión y visualización de una librería en línea.

## Estado del Proyecto

- **ApiGateway**: Funcional (Hay que implementar seguridad)⚒️
- **EurekaServer**: Finalizado ✅
- **ConfigServer**: Finalizado ✅
- **BookMicroservice**: Finalizado (Test incluidos) ✅
- **UserMicroservice**: En desarrollo ⏳
- **KafkaMicroservice**: Pendiente ⏳
- **OrderMicroservice**: Pendiente ⏳
- **PaymentMicroservice**: Pendiente ⏳

## Descripción

Este proyecto está diseñado para proporcionar una interfaz intuitiva y funcional para administrar libros, realizar búsquedas por título, autor, género literario y otros criterios. Además, permite gestionar inventarios, disponibilidad de libros y realizar operaciones CRUD (Crear, Leer, Actualizar, Eliminar).
