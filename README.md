# LibreriaOnline

Librería Online es una plataforma en desarrollo diseñada para ofrecer una solución integral de gestión y visualización de una librería en línea. Este proyecto está orientado a dos tipos de usuarios principales:

Administradores :
Los administradores tendrán acceso a herramientas avanzadas para gestionar el catálogo de libros, incluyendo la capacidad de agregar, actualizar y eliminar productos, así como supervisar el inventario, gestionar pedidos y personalizar la configuración de la plataforma.

Usuarios Finales :
Los usuarios podrán explorar el catálogo de libros con múltiples opciones de búsqueda y filtrado, crear listas de deseos, guardar libros para futuras compras y realizar transacciones seguras en línea desde la comodidad de su hogar.
Con un enfoque en la usabilidad y la eficiencia, Librería Online busca brindar una experiencia fluida tanto para los administradores como para los clientes finales, combinando funcionalidades robustas con una interfaz intuitiva y moderna.
